About The Project

Tandem is an open source project that was started in May of 2018 by Giancarlo Coelho, Peter Soboyejo, and Alex Trevino. This project aims to be an open source multiplayer game that focuses on the drivers ability to control their car at an angle, or a drift. The user selects their car, and joins online servers racing against others.

About The Community

Becuase Tandem is open-source, we depend heavily on what those in the community using our project think about it. You too can help out by letting us know different ways we can improve! If you want to, please feel free to visit our linked GitHub page and download the source code for yourself. From there, set up an enviroment and play aroud how ever you would like to! We are very open to suggestions so don't be afraid to ask up to add anything! Constant updates will be pushed, so each week expect new and exciting content and bug fixes to be added.
