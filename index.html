About The Project

Tandem is an open source project that was started in May of 2018 by Giancarlo Coelho, Peter Soboyejo, and Alex Trevino. This project aims to be an open source multiplayer game that focuses on the drivers ability to control their car at an angle, or a drift. The user selects their car, and joins online servers racing against others.
